# Problem
    Given a string that represents time in hours and minutes, convert the string to military time (if necessary).

# Examples
    Input        Output
    "3:00pm"     "15:00"
    "9:15am"     "09:15"
    "12:00am"    "00:00"
    "04:00"      "04:00"
    "11:45pm"    "23:45"
    "1:23am"     "01:23"
    "12:01am"    "00:01"
    "7:47pm"     "19:47"
    "12:00pm"    "12:00"
